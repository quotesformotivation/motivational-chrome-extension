# Quotes for Motivation: Focus & Productivity Hack

A lightweight Chrome extension that surfaces non-intrusive motivational quotes while you work. Designed to boost focus without hijacking your New Tab.

**Official Website:** [Quotes For Motivation](https://quotesformotivation.com)

## Features

- Smart quote scheduling with category filters
- Dark mode–aware, minimal UI
- Fast, offline-friendly quote delivery
- Clean overlay with share actions

## Install (Developer Mode)

1. Open `chrome://extensions/` in Chrome.
2. Enable **Developer mode** (top right).
3. Click **Load unpacked** and select this folder.

## Project Structure

```
assets/     # icons + quote datasets
background/ # service worker + scheduling
content/    # overlay injection
popup/      # extension popup UI
manifest.json
```
